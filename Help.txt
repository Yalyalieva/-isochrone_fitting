Горячие клавиши:

A Z - сдвиг изохроны по оси ординат
, . - сдвиг изохроны по оси абсцисс
N P - изменение возраста изохроны

[ ] - изменение шага по оси оридинат
; ' - изменение шага по оси абсцисс

( ) - изменение количества точек на графике плотности


Файл с данными должен содержать три столбца: расстояние звезды от центра скопления, звездная величина и цвет.
У файла не должно быть заголовка с названием столбцов (или они должны быть закомментированы '#', как в примере 'ngc225_jjh.csv').

Загрузка новых изохрон.

1. Нажмите кнопку "Загрузить новые теоретические изохроны".
2. В появившемся диалоговом окне с помощью кнопки "Загрузка" выберете файл.
Структура файла: 3 стоблца, первый - logt, второй - звездная величина, третий - цвет.
В файле не должно быть названий столбцов, либо они должны быть закомментированы знаком '#'.
3. Заполните данные об изохронах.

Пример: требуется загрузить изохроны (J,J-H) с металличностью 0.02, 
соответсвующие фильтрам обзора 2MASS. Следует заполнить графы следующим образом:
Фильтр J
Цвет J-H
Система фильтров 2mass
Металличность 0.02
